# Checklist Checklist

## Pre-Generation

- [ ] Identified the artifact kind
- [ ] Know the template structure
- [ ] Know the content requirements
- [ ] Understand generation workflow
- [ ] Understand validation workflow

## Structure

- [ ] Has Pre-Generation section
- [ ] Has Structure section
- [ ] Has Content section
- [ ] Has Post-Generation section
- [ ] Has Quality section

## Content

- [ ] Pre-Generation items prepare for work
- [ ] Structure items match template sections
- [ ] Content items cover requirements
- [ ] Post-Generation items verify completeness
- [ ] Quality items ensure standards

## Post-Generation

- [ ] All sections have items
- [ ] Items are checkbox format
- [ ] Items are verifiable (yes/no)
- [ ] No duplicate items
- [ ] No placeholder text

## Quality

- [ ] Items are actionable
- [ ] Items are in logical order
- [ ] Language is consistent
- [ ] Coverage is complete
