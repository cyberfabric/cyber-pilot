# Design Review Prompt

Review the following PR changes against design and architecture best practices.

Focus on:
- Alignment with existing architecture (see docs/ARCHITECTURE_MANIFEST.md)
- Trade-off analysis
- API contract consistency
- Security considerations
- Compliance with /docs/spec-templates/DESIGN.md template
- Find antipatterns
- Compare proposed design with existing industry patterns in SaaS platforms
- Compare proposed design with the best IEEE, ISO, and other industry standards
- Criticize the design
- Split by categories and rate every 1-10

Use `docs/checklists/DESIGN.md` as the structured review guide.
Pay attention to the "PR Review Focus (Design)" section at the end of the checklist.
